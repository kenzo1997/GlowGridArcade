# 🕹️ GlowGrid Arcade

A retro-inspired, neon-drenched web arcade built with **Phaser 3**. This project features a collection of classic arcade and board games accessible through a centralized "Game Hub" interface.

## 🚀 Featured Games

The arcade currently hosts six unique gaming experiences:

* **Tic-Tac-Toe**: Classic strategic grid combat.
* **Four In A Row**: Gravity-based "Connect Four" with a pulsing winning highlight.
* **Frogger**: High-stakes road and river crossing. (not working)
* **Mine Dash**: A fast-paced infinite runner with double-jump mechanics. (not working)
* **Battleship**: A tactical "Search & Destroy" game against a CPU AI. (WIP)
* **Dodge & Catch**: A frantic reflex game—catch gems, avoid bombs.

## 🛠️ Tech Stack

* **Engine:** [Phaser 3](https://phaser.io/) (JavaScript Game Framework)
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), and Vanilla JavaScript
* **Styling:** Custom "CRT" scanline effects and Neon-morphic UI components
